The TabLift.dmg is created by using the following command:
```bash
   create-dmg \
  --volname "TabLift Installer" \
  --window-pos 200 120 \
  --window-size 835 650 \
  --icon-size 100 \
  --hide-extension "TabLift.app" \
  --background "tablift-bg.png" \
  --icon "TabLift.app" 250 250 \
  --app-drop-link 600 250 \
  "TabLift.dmg" \
  "TabLift.app"
  ```
